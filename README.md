# Voronoi_Diagram
Creating Voronoi Diagramm in Python

## Getting started
Generate the Voronoi Cell for a reference point.

## Prerequisites
* Python 3.* 
* Matplotlib 

## Installing
Run `cell.py` to launch the program.
